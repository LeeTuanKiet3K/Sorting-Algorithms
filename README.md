# Sorting-Algorithms
This is the source code of 11 sorting algorithms - by students from HCMUS
Compile: g++ main.cpp -o group07.exe
Command line arguments (Note: create "input.txt" for command 1 and command 4):
- Command 1: group07.exe -a radix-sort input.txt -both
- Command 2: group07.exe -a selection-sort 100 -rand -time
- Command 3: group07.exe -a merge-sort 7000 -comp
- Command 4: group07.exe -c heap-sort merge-sort input.txt
- Command 5: group07.exe -c binary-insertion-sort insertion-sort 100000 -nsorted
